# node-observability

Node.js による可観測性の機能テスト

## Feature

- OpenTelemetry（中間基盤）
- Jaeger（トレース可視化）
- Prometheus（メトリクス収集）
- Grafana（ダッシュボード表示）

## Requirement

| Language/FrameWork | Version |
| :----------------- | ------: |
| TypeScript         |   5.8.3 |
| Node.js            | 22.16.0 |
| Docker Engine      |  28.1.1 |
